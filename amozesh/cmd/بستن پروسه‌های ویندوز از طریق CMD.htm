بستن پروسه‌های ویندوز از طریق CMD

همان  طور که می دانید برای بستن پروسه های ویندوز بایستی به Task Manager و سپس تب Processes مراجعه نمایید. اما شاید برای تان جالب باشد که بدانید این کار را می توانید از طریق Command Prompt ویندوز نیز انجام دهید، بدون اینکه به نیاز به استفاده از Task Manager داشته باشید. در این ترفند به معرفی نحوه انجام این کار می پردازیم.

 

 

بدین منظور:

 

ابتدا کلیدهای ترکیبی Win+R را فشار دهید تا پنجره Run باز شود.

 

سپس عبارت cmd را وارد کرده و Enter بزنید.

 

پس از باز شدن Command Prompt در خط فرمان این دستور را را وارد نمایید:

 

tasklist

 

با این کار لیست کامل پروسه های در حال اجرا در آن لحظه نمایش داده خواهد شد.

 

حال برای بستن یک پروسه این دستور را وارد نمایید:

 

taskkill /pid PID

 

به جای PID بایستی شماره ای که در ستون PID هر پروسه وجود دارد را وارد نمایید.

 

به عنوان مثال اگر پس از استفاده از دستور tasklist، در ستون PID پروسه ی firefox.exe عدد 75740 درج شده بود، از این دستور برای بستن پروسه ی firefox.exe استفاده خواهیم کرد:

 

taskkill /pid 75740

 

دقت کنید اگر پس از این کار با پیغام  خطای ERROR: The process with PID # could not be terminated روبرو شدید برای بستن اجباری این پروسه بایستی از دستور زیر استفاده نمایید:

 

taskkill /pid PID /f
