ایجاد منوی بازشو یا کشویی (DropDown) و اعتبارسنجی با PHP

در حین ساخت فرمهای HTML، یکی از مواردی که ممکن است به آن نیاز داشته باشید، فیلدهای منویی بازشو یا Drop Down هستند. اینگونه فیلدها معمولا به این صورت میباشند که با کلیک بر روی آنها، لیستی از انتخابات موجود برای شما نمایش داده میشود و میتوانید یک گزینه را انتخاب کنید. در این مطلب قصد داریم نحوه ی ساخت یکی از این فرمها را در HTML و اعتبارسنجی آن را در PHP آموزش دهیم.

پیش از شروع آموزش میتوانید به مطلب زیر مراجعه کنید تا اطلاعات جامعی در خصوص فرمهای HTML کسب کنید:

» آموزش جامع ساخت فرمهای HTML

پیش نیازها:

  1. داشتن اطلاعات کافی از HTML و کار با تگ form و عناصر آن
  2. داشتن اطلاعات مقدماتی از PHP

ساخت منوی کشویی (<select>) در HTML

برای ساخت هر فرم در html باید از تگ <form> استفاده کنیم و برای ایجاد منوی کشویی (بازشو) باید از تگهای select و option استفاده کنیم. در این آموزش یک فرم ساده ایجاد میکنیم که کاربر بین دو گزینه ی “مرد” و “زن” حق انتخاب دارد و گزینه ی پیشفرض نیز عبارت “انتخاب کنید” است.


<form action="process.php" method="post" >
<select name="Select_Name">
<option value="">انتخاب کنید</option>
<option value="M"> مرد</option>
<option value="F"> زن </option>
</select>
<input type="submit" value="ارسال" />
</form>

یک تکه کدکوتاه که البته نیازمند توضیح است.

مانند هر فرم دیگری، برای اینکه اطلاعات فرم پردازش شود باید از مشخصه ی action استفاده کنیم و آدرس صفحه ی پردازنده را در آن وارد کنیم. پس action=process.php به این معنی است که اطلاعات فرم به صفحه ای با آدرس process.php ارسال میگردد تا در آنجا پردازش شود. با استفاده از method نیز تعیین میکنیم نحوه ی ارسال و دریافت اطلاعات به چه گونه باشد. بهتر است برای فرم ها از متد Post استفاده کنید مگر در موارد معدود!

در مرحله ی بعدی با استفاده از تگ <select> یک فرم بازشو (کشویی) ایجاد کردیم. باید نام منحصر به فردی به این تگ داده شود. برای همین منظور با استفاده از مشخصه ی name یک نام خاص به فیلد select دادیم. (name=Select_Name). میتوانید از هر نام دیگری که دلخواه شما است استفاده کنید فقط دقت داشته باشید که این نام نباید تکراری باشد.

با استفاده از تگهای <option> موارد تشکیل دهنده ی منو را ایجاد کردیم و برای هر option یک مقدار یا Value درنظر گرفتیم. این مقادیر برای هرکدام از optionها باید جداگانه باشند و تکراری نباشند. برای مثال برای option مرد مقدار M و برابر option زن مقدار F را در نظر گرفتیم و برای option انتخاب کنید هیچ مقداری را در نظر نگرفتیم و در واقع آنرا برابر “” (تهی) قرار دادیم. بدین ترتیب میدانیم که کاربر کدام یک از آیتم های موجود در لیست را انتخاب کرده است.

در انتها نیز یک دکمه برای ارسال اطلاعات فرم تعبیه میکنیم.

اما نحوه ی دریافت اطلاعات فرم کشویی (بازشو) و پردازش اطلاعات آن در PHP به چه صورت است؟

دریافت و پردازش اطلاعات فیلد select در فرم html با استفاده از PHP

دریافت اطلاعات فیلد select در php نیز مانند هر فیلد دیگری است. اگر دقت کرده باشید برای فیلد select یک name انحصاری انتخاب کرده بودیم. در اینجا name=Select_Name می باشد. در php کافی است اطلاعاتی که در این نام ذخیره شده است را بازیابی کنیم:


<?php
$select = $_POST['Select_Name'];
?>

در قطعه کد بالا به صورت ابتدایی اطلاعات فیلد select را دریافت کردیم و داخل متغیری با نام select$ ذخیره کردیم. حال برای اعتبار سنجی میتوانیم از متغیر select$ استفاده کنیم.

اعتبارسنجی به شیوه های مختلفی میتواند انجام شود که برخی از آنها را نام میبریم:

  1. موردی از لیست انتخاب نشده باشد (گزینه ی “انتخاب کنید” انتخاب شده باشد)
  2. گزینه ی مرد انتخاب شده باشد
  3. گزینه ی زن انتخاب شده باشد

اکنون که مقدار تکمیل شده توسط کاربر را از فیلد Select در اختیار داریم میتوانیم با استفاده از دستورات شرطی اعتبارسنجی را انجام دهیم:

اعتبارسنجی: حتما باید یک مورد انتخاب شود

اگر فیلدی انتخاب نشده باشد، پس فرم بر روی گزینه ی “انتخاب کنید” قرار خواهد داشت و مقدار value برای این فیلد یک عبارت تهی ( ” “) است.

به عبارت ساده تر، اگر مقدار select$ برابر تهی باشد یعنی موردی انتخاب نشده است:


if (empty($select)){
echo " گزینه ای انتخاب نشده است";
}

در صورتی که بجای دستور echo از دستور die استفاده کنیم، اجرای کدها متوقف شده و ادامه ی کدها نمایش داده نخواهند شد.

اگر بخواهیم مقدایر موجود در لیست را اعتبارسنجی کنیم نیز به همین ترتیب عمل میکنیم. یعنی مقداری که به value یک option داده ایم را با مقدار متغیر select$ مقایسه میکنیم. برای مثال:


if($select == "M"){
echo "<h1>گزینه ی مرد انتخاب شده است.</h1>";
}
if($select == "F"){
echo "<h1>گزینه ی زن انتخاب شده است.</h1>";
}

با استفاده از دستور شرطی if تعیین کردیم که اگر مقدار select$ برابر با M باشد کاربر گزینه ی مرد را انتخاب کرده است و اگر برابر با F باشد کاربر گزینه ی زن را انتخاب کرده است. (M در فیلد option به عنوان value برای گزینه ی مرد در نظر گرفته شده بود و F به عنوان value برای گزینه ی زن).

به این ترتیب کد اعتبارسنجی ما به این صورت خواهد بود:


<?php
$select = $_POST['Select_Name'];
if(empty($select)){
echo "<h1>گزینه ای انتخاب نشده است.</h1>";
}
if($select == "M"){
echo "<h1>گزینه ی مرد انتخاب شده است.</h1>";
}
if($select == "F"){
echo "<h1>گزینه زن انتخاب شده است.</h1>";
}
?>
